# 🎤 HIKAKIN 豆知識クイズ｜10問徹底解説WEBサイト

ユーザー提供のテキストファイル「🎤 ヒカキンに関する豆知識クイズ 10問.txt」（21,714 bytes）の本文を **一字一句改変せず** 完全収録した、和モダン × プレミアム設計の高品質クイズWebサイト。

---

## ✅ 実装済み機能

### 1. ヒーロー＆ナビゲーション（`index.html`）
- 和モダンプレミアム配色（深いインディゴ／純金／朱／和紙のオフホワイト）  
  ※ **ネオン系の蛍光色・サイバーカラーは一切不使用**
- アニメーション付き「日輪」「スキージャンプ軌跡」「浮遊する音符」をSVGで自作
- 回転する金色エンブレム（マイクSVG + 二重リング）
- 統計表示 / 2段CTA
- IntersectionObserverによるスクロール時のフェードイン

### 2. クイズ本体（`quiz.html`）
- 全10問を**1ページSPA形式**で進行
- **3Dカードフリップアニメーション**による問題切り替え
- 選択肢ホバーで**金箔シマー効果**
- 単一選択：クリック即判定 / 複数選択（Q6, Q10）：チェック→「回答する」ボタン
- 正解時：**Canvas紙吹雪**（金・朱・抹茶・桜の落ち着いた配色）＋金色リング波紋
- 不正解時：**カードシェイク** + 上品な低音
- 解説パネルは**和紙テクスチャ風オフホワイト**でカードから展開
- 解説には正解理由・引用元・不正解解説・テーブル表示・補足リストを完備
- **Web Audio APIによる合成効果音**（外部音源ファイル不要）
- キーボードショートカット：A〜Eで選択、Enterで次へ
- **進捗バーは使用せず**、代わりに10個のドットインジケータでステータス表示

### 3. 結果＆ランキング（`result.html`）
- 6段階のランク（S/A/B/C/D/E）と称号 + 専用コメント
- 金色のクレスト（紋章）＋回転リング
- 10問の正誤マトリクス
- **localStorage** によるスコア保持＋プレイヤー名記憶
- **RESTful Table API** `quiz_scores` への登録機能
- 上位20名のランキング表示（メダルカラー付き、自分の記録ハイライト）

### 4. 全問解説アーカイブ（`archive.html`）
- 全10問の本文・選択肢・正解・解説をフル収録
- 上部に問題ジャンプリンク（TOC）
- 各問の正解選択肢を緑で明示
- ファイル末尾の「おまけ豆知識」「クロージング文」もそのまま掲載

### 5. ヒカキン年表（`about.html`）
- 1989年の誕生〜2026年の麦茶ブランド「ONICHA」まで全16イベント
- 中央線＋ドットの**左右交互タイムライン**
- スクロールでカードが順次フェードイン
- 末尾に原文クロージング引用

## 🌐 主要URI

| パス | 説明 |
|---|---|
| `/` または `/index.html` | トップページ |
| `/quiz.html` | クイズ本体（全10問） |
| `/result.html` | 結果＆ランキング |
| `/archive.html` | 全問解説アーカイブ |
| `/about.html` | ヒカキン年表 |

### RESTful Table API（自動提供）
| メソッド・パス | 内容 |
|---|---|
| `GET tables/quiz_scores` | ランキング一覧取得 |
| `POST tables/quiz_scores` | 新規スコア登録 |

### `quiz_scores` テーブルスキーマ
| フィールド | 型 | 説明 |
|---|---|---|
| `id` | text | レコードID（自動付与） |
| `name` | text | プレイヤー名（ニックネーム） |
| `score` | number | 正解数（0〜10） |
| `total` | number | 総問題数（10） |
| `duration_sec` | number | 所要時間（秒） |
| `timestamp` | number | 完了タイムスタンプ（ミリ秒） |

## 🎨 デザインの方針

- **配色**：深いインディゴ `#1a2238` ベース、純金 `#d4af37`、朱 `#c5392e`、和紙 `#f7f3ec`
- **タイポ**：Noto Serif JP（見出し）、Noto Sans JP（本文）、Bebas Neue（数字・英語ラベル）
- **背景**：和柄「麻の葉」をSVGでパターン化した極薄テクスチャ
- **解説パネル**：和紙のざらつきをSVGノイズで再現
- **演出**：
  - 紙吹雪（純Canvas、外部ライブラリ不要）
  - 金色リング波紋
  - カードフリップ＆シェイク
  - 金箔シマー（hover時に光が流れる）
  - 浮遊する音符・回転するエンブレム

## 🚫 ユーザー指示で守ったルール
